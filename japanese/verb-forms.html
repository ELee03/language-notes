Verb Forms

The forms grammar patterns attach to. This is the missing bridge between a dictionary verb and actual dialogue.

Verb Classes

食たべる -> 食たべます -> 食たべない -> 食たべた
먹다 -> 먹습니다 -> 먹지 않는다 -> 먹었다
ichidan: to eat -> polite -> negative -> past
読よむ -> 読よみます -> 読よまない -> 読よんだ
읽다 -> 읽습니다 -> 읽지 않는다 -> 읽었다
godan: to read -> polite -> negative -> past

How To Read A Grammar Card

Pattern tells you the visible grammar. Attaches to tells you the form that must come immediately before it. If that field says “verb ます-stem + たい,” then 行く becomes 行きたい, not 行くたい.
